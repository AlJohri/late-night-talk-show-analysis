## Usage

Update latest `LATEST_EPISODE_NUMBER` in `scripts/settings.py`.

1. `make data` - download latest data
2. `make notebook` - open jupyter notebook
3. `make gist` - upload latest jupyter notebook to gist and open nbviewer
